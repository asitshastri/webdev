Emmet Shortcuts

  1. ctrl + /: comment
  2. alt + d: line copy down
  3. alt + mouse select: multiple target for kleyboard input
  4. tagname*n: n no. of mentioned tags
  5. tagname.class1: to define a tag with class 1
  6. tagname.class1.class2......classn: to define mentioned tag with multiple classes
  7. tagname.class1.class2......classn*n: to define n no. of tags with multiple classes
  8. tagname#id: to define a tag with an id
  9. tagname.class1.class2#id1: define a tag with class 1 ,class2 and id1
